FAQ

Project Overview?

The Scale overview of this design was to create a mobile application that allowed users to view different biomes, and learn about the animals that live within it.

Ecountered Problems?

Making sure that the application itself was working on numerous screens was a problem I encountered, but solved it overtime. In addition, making sure all links worked and efficiently transferred the user to the next page was a problem, which I once again overcome gradually.

Solution & Goal?

Target Audience: This project aims towards adults and Young People. This is due to the way the application is based on teaching people new things, being animals. People who are looking to get into the workplace of the animal industry would certainly take a liking into it. A common goal was to make sure the design flowed throughout the entire design as a whole, making sure you were able to feel the application flow throughout the design. I did this by creating seamless animations within buttons, that felt surreal when pressing it. In addition, I also wanted to keep the layout similar, whilst not making the application as a whole feel like its for children, but instead so that it feels like a design pattern everyone can use. I managed to do this by creating simple yet effective design patterns, whilst keeping the layout to a mature standard!.
